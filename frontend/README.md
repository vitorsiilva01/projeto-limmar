Frontend (atual)

Este repositório contém a aplicação frontend em React (Vite).
Criei a pasta `frontend/` para separar responsabilidades. No momento o código existente está na raiz `src/` — você pode mover esses arquivos para a pasta `frontend/src/` se desejar uma separação completa.

Sugestão rápida para mover:
1. Crie a pasta `frontend/src/` e mova `src/` para lá (ou copie).
2. Atualize os caminhos relativos e `package.json` scripts caso queira executar o frontend dentro da pasta `frontend`.

Por enquanto, mantenha o projeto atual conforme está. A pasta existe como marcador para separar frontend/backend.
